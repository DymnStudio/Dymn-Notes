# ✨ Features

- 📝 Create and edit notes
- ✅ Interactive checklist notes
- ⭐ Favorite notes
- 📦 Archive notes
- 🔍 Instant search
- 🎨 Custom wallpapers
- 🎨 Custom tile colors
- 🌙 Dark theme
- ⚡ Fast and lightweight
- 📱 Material Design 3
